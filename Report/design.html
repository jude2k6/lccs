Website

The website should display the graphs in a way that makes them accessible to all users possible so they should have some built in accessibility features such as an accessible font such as comic Sans which makes it easy for people with dyslexia to read and text to speech if possible. The website also has to be able to advise users based on an input using the data from the dataset this should have error catching so users can not input values that would break the program, with my chosen dataset I think a good way to show this would be to have a user input their salary, age, gender and level of education it would then process this data and tell users if they are being paid fairly compared to those with similarly factors. The site should also gather some data from users, store it and display it on the website in some way. I think the easiest way to achieve this would be to store comments in a json file as python works with json files very easily.
